# coder-notifier

> 一款定时提醒工具，如果你经常忘记喝水休息，这很管用。
---------
[![npm](https://img.shields.io/npm/dm/ink?logo=npm)](https://npmjs.com/package/coder-nofitier)

## 安装

```bash
$ npm install --global coder-notifier
```

## 快速开始

```bash
# 帮助文档
$ coder-notifier --help

# 设置您的通知
$ coder-notifier set

# 启动
$ coder-notifier start
```

## 高级用法

```bash
# 持久化运行，开机自启动。
$ coder-notifier save

# 取消持久化运行。
$ coder-notifier stop

# 关于我们
$ coder-notifier about
```
